## 1. Кнопка «📞 Заказать»

FSM `OrderRequestFSM` в client_bot.

**Шаг 1 — Выбор услуги**
```
📞 Заявка на заказ

Выберите услугу:
[Уборка квартиры]  [Мойка окон]
[Химчистка дивана] [Другое]
[❌ Отмена]
```
Кнопки генерируются из справочника мастера (`services` где `is_active = true`).
«Другое» — клиент вводит текстом.

**Шаг 2 — Комментарий**
```
Добавьте комментарий (необязательно):
Например: площадь квартиры, адрес, пожелания

[Пропустить]  [❌ Отмена]
```

**Шаг 3 — Подтверждение**
```
📋 Ваша заявка:

🛠 Услуга: Уборка квартиры
💬 Комментарий: 3-комнатная, 80 м²

[✅ Отправить заявку]  [❌ Отмена]
```

**После отправки:**
1. Сохранить в `inbound_requests` (тип `order_request`)
2. Отправить уведомление мастеру в master_bot:
```
🛎 Новая заявка на заказ!

👤 {имя клиента}
📞 {телефон}
✈️ @{username если есть}
🛠 Услуга: {услуга}
💬 {комментарий если есть}
```
3. Клиенту показать:
```
✅ Заявка отправлена!

Мастер свяжется с вами в ближайшее время.
📞 {контакты мастера}
```
Кнопка: `[🏠 Главная]`

## 2. Кнопка «❓ Вопрос»

FSM `QuestionFSM` в client_bot.

**Шаг 1 — Текст вопроса**
```
❓ Задать вопрос мастеру

Введите ваш вопрос:
[❌ Отмена]
```

**После отправки:**
1. Сохранить в `inbound_requests` (тип `question`)
2. Отправить уведомление мастеру в master_bot:
```
❓ Вопрос от клиента

👤 {имя клиента}
📞 {телефон}
✈️ @{username если есть} → tg://user?id={tg_id}

Вопрос:
{текст вопроса}
```
Username в уведомлении — кликабельная ссылка на клиента чтобы мастер мог ответить напрямую в личку.

3. Клиенту показать:
```
✅ Вопрос отправлен!

Мастер ответит вам в личные сообщения.
```
Кнопка: `[🏠 Главная]`

## 3. Кнопка «📸 Фото/видео»

FSM `MediaFSM` в client_bot.

**Шаг 1 — Медиа**
```
📸 Отправить фото или видео мастеру

Например: фото объекта для оценки стоимости,
видео поломки, фото до/после

Отправьте фото или видео:
[❌ Отмена]
```
Принимать: `photo`, `video`, `document` (на случай если клиент отправит файл).

**Шаг 2 — Комментарий**
```
Добавьте комментарий (необязательно):

[Пропустить]  [❌ Отмена]
```

**После отправки:**
1. Сохранить в `inbound_requests` (тип `media`, сохранить `file_id`)
2. Переслать медиа мастеру в master_bot с подписью:
```
📸 Медиа от клиента

👤 {имя клиента}
📞 {телефон}
✈️ @{username если есть}
💬 {комментарий если есть}
```
Медиа пересылать через download + re-upload (тот же паттерн что в рассылке).

3. Клиенту показать:
```
✅ Отправлено!

Мастер получил ваше фото/видео.
```
Кнопка: `[🏠 Главная]`

## Критерии проверки

1. «Заказать»: услуги из справочника мастера подгружаются динамически
2. «Другое» — клиент вводит услугу текстом
3. Мастер получает уведомление о заявке с данными клиента
4. «Вопрос»: уведомление мастеру содержит кликабельную ссылку на клиента
5. Мастер может ответить клиенту напрямую нажав на ссылку в уведомлении
6. «Фото/видео»: принимает photo, video, document
7. Медиа доставляется мастеру (download + re-upload)
8. Все три запроса сохраняются в inbound_requests
9. Если мастер заблокировал бота — перехватить ошибку, клиенту показать сообщение
10. После каждого действия клиент возвращается на Home кнопкой
